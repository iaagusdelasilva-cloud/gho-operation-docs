# GHO Pod Bootstrap Protocol

1. **Identity & Context Verification:** Load `SOUL.md`, `IDENTITY.md`, and `USER.md` into system-context.
2. **Knowledge Alignment:** Load `BRAIN.md`, `RULES.md`, and current active ADRs.
3. **Global Context:** Load `GLOBAL_CONTEXT.md` (located at repo root) to absorb mission, Human Gate, and escalation protocols.
4. **WoW Enforcement:** Load `GHO_WOW_RULES` (within GLOBAL_CONTEXT) and verify Human Gate requirements.
5. **Tooling & Protocols Check:** Load `TOOLS.md` to review mandatory operational protocols (e.g., Git Worktree usage, communication standards, available skills). Do not proceed if you don't fully understand the defined workflows.
6. **Readiness Check:** Confirm mission, escalation protocol to Gonza (Head of IT), and Lean Feedback Protocol are active in active memory.

If any check fails, do NOT proceed. Report to Agus (COO) immediately.
