# Obsidian Vault for GHO Knowledge Base
This vault is the **human interface** for GHO's tactical and strategic knowledge.

## Rules of Engagement
1. **Sync:** All files in this folder are synchronized with the `gho-knowledge-base` repository.
2. **Atomic Notes:** Write notes as standalone concepts (Kebab-case filenames).
3. **Template-Driven:** Use the templates provided in `templates/` for ADRs, Meeting Minutes, and Brainstorming.
4. **No Secrets:** Do NOT store raw API keys or sensitive credentials here. 
5. **Human-First:** This is your space. Don't worry about machine readability yet; the `Memory Curator` will destill your notes into `.psv` passages later.

## Structure
- `/daily-notes`: Your chronological journal.
- `/projects`: Specific GHO project contexts.
- `/templates`: Official GHO templates for rapid drafting.
